Banking ETL
Pipeline

Python PostgreSQL Pandas SQLAlchemy Airflow

Visão geral

Pipeline automatizado para extração, transformação e carga de dados bancários. Processa mais de 2 milhões de registros diários, com validação de dados, tratamento de erros e logging completo.

→ 2.4M registros/dia
→ Redução de 70% no tempo de processamento
→ 99.8% de confiabilidade nos dados
→ Alertas automáticos via email em caso de falha

Como funciona

O pipeline roda diariamente via Airflow. A etapa de extração puxa dados de múltiplas fontes (APIs e arquivos CSV legados), a etapa de transformação aplica regras de negócio e validações, e a carga insere os dados limpos no data warehouse PostgreSQL.
