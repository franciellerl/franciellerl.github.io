// visão_geral // overview

Simulação de um sistema multiagentes aplicado à exploração de ambientes desconhecidos, desenvolvida para a disciplina de Inteligência Artificial. O projeto recria uma comunidade de agentes autônomos responsáveis por explorar um planeta, coletar recursos naturais e transportá-los até uma base central, aplicando conceitos como arquitetura BDI, busca em grafos e tomada de decisão por utilidade.

A multi-agent system simulation applied to unknown environment exploration, developed for an Artificial Intelligence course. The project recreates a community of autonomous agents responsible for exploring a planet, collecting natural resources, and transporting them to a central base, applying concepts such as BDI architecture, graph search, and utility-based decision making.

→ Ambiente gerado dinamicamente com obstáculos e recursos distribuídos
→ Três arquiteturas de agentes: Reativo, Baseado em Estado e BDI
→ Navegação com BFS e crenças compartilhadas entre agentes
→ Interface interativa em tempo real com Streamlit → Dynamically generated environment with obstacles and distributed resources
→ Three agent architectures: Reactive, State-Based, and BDI
→ BFS navigation and shared beliefs between agents
→ Real-time interactive interface built with Streamlit

// estrutura_do_projeto // project_structure

01
models.py — Modelagem do Ambiente Environment Modeling

Representação do planeta e distribuição dos recursos disponíveis. Cria o mapa de exploração, posiciona a base central, gera obstáculos naturais como montanhas e rios, e implementa controle de movimentação dos agentes. A navegação utiliza Breadth-First Search (BFS) para identificar caminhos válidos e monitorar recursos ao longo da execução.

Planet representation and distribution of available resources. Creates the exploration map, positions the central base, generates natural obstacles such as mountains and rivers, and handles agent movement control. Navigation uses Breadth-First Search (BFS) to identify valid paths and monitor resources throughout the simulation run.

02
agents.py — Desenvolvimento dos Agentes Agent Development

Define os agentes exploradores e seus mecanismos de tomada de decisão. Uma classe base Agent fornece funcionalidades comuns como controle de inventário, entrega de recursos e cálculo de utilidade. A partir dela, são implementadas três arquiteturas distintas: agente reativo, baseado em estado e BDI — com compartilhamento de crenças coletivas entre agentes.

Defines the explorer agents and their decision-making mechanisms. A base Agent class provides shared functionality such as inventory management, resource delivery, and utility calculation. From it, three distinct architectures are implemented: reactive, state-based, and BDI — with collective belief sharing across agents.

03
app.py — Interface e Execução da Simulação Interface and Simulation Execution

Responsável pela inicialização, execução e visualização da simulação. A interface gráfica em Streamlit permite configurar os parâmetros antes de rodar, acompanhar os agentes em tempo real no mapa e visualizar estatísticas de desempenho ao final de cada ciclo de simulação.

Handles initialization, execution, and visualization of the simulation. The Streamlit interface allows configuring parameters before running, tracking agents in real time on the map, and viewing performance statistics at the end of each simulation cycle.
